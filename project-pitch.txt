# Track it, Lose it - Project Pitch

## Description

Track it, Lose it is a platform designed to assist users in tracking their fitness journey and achieving their weight loss goals. The platform allows users to record their weight, track their exercises, calculate their Total Daily Energy Expenditure (TDEE), and discover recipes that align with their specific calorie range.

## Background

Track it, Lose it was initiated as a personal project by its creator. The aim was to create a resource that could conveniently track weight and exercise data on a daily basis. The project evolved into Track it, Lose it to meet this need more effectively.

## MVP Features

- Personalized User Page: Users have access to a personalized page where they can view their favorited recipes, exercise notes, and a chart illustrating their weight progress.

- Weight Tracking: Users can submit their weight notes, enabling them to reference them later. This feature allows users to track the body groups they have exercised during the week and identify those they haven't.

- TDEE Calculation: Users can obtain an accurate estimation of their Total Daily Energy Expenditure (TDEE) based on their weight, age, height, and activity level. Track it, Lose it utilizes the Mifflin-St Jeor Equation for precise calculations.

- Weight Progress Visualization: Users can monitor their weight progress over time through an interactive line graph. This visualization helps users track their weight changes and identify trends.

- Recipe Search: Users can search for recipes based on their desired minimum and maximum calorie limits. Track it, Lose it provides recipe recommendations that fit within the specified calorie range, along with detailed ingredient lists and instructions.

## Tech Stack

- **Database:** PostgreSQL
- **Backend:** Python 3
- **Frontend:** HTML, CSS, Bootstrap, JavaScript for DOM manipulation

## Dependencies

- Python packages:
  - SQLAlchemy ORM
  - Flask
  - Jinja
- APIs/External Data Sources:
  - Spoonacular API
- Browser/Client-Side Dependencies:
  - Bootstrap
  - React

Track it, Lose it empowers users to conveniently track their weight, exercise progress, and dietary choices within a single platform, aiding them in their weight loss journey and fostering a healthier lifestyle.
